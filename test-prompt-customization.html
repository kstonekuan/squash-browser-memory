Manual Test Plan for Prompt Customization

Test Steps:

  1. Start the development server with pnpm run dev
  2. Open the application in your browser
  3. Enter your API key
  4. Scroll down to the bottom of the page
  5. Click on "Customize Prompts" to expand the section
  6. Verify you can see two textareas with the default prompts
  7. Upload or paste some Chrome history data
  8. Click analyze and verify the "Intermediate Standardized Format" section appears
  9. Click on "Intermediate Standardized Format" to expand it
  10. Verify you can see:
    • Statistics (Total URLs, Date Range, Unique Domains)
    • Top Domains list
    • Raw Standardized Data in JSON format
  11. Go back to "Customize Prompts" and modify the prompts
  12. Run another analysis and verify the modified prompts are used
  13. Click "Reset to Defaults" and verify the prompts reset

Expected Behavior:

  • The intermediate standardized format should be visible after the first LLM call
  • Custom prompts should be applied to subsequent analyses
  • The reset button should restore the default prompts
